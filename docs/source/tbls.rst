

Command-line Arguments
======================


This section summarizes arguments for the four commands described in the other sections. 

.. list-table:: ``recommend_annotation``
   :widths: 25 25 50 25 
   :header-rows: 1

   * - name
     - type
     - description
     - default value
   * - model
     - string
     - SBML model file
     - N/A
   * - cutoff
     - float (0.0 - 1.0)
     - match score cutoff
     - 0.0
